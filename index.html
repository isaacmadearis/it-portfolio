The path into cloud

I'm a Salesforce Certified Administrator moving into cloud computing and IT support. Years of building flows, dashboards, and CRM fixes showed me how much I enjoy setting up systems that make people's jobs easier.

That pulled me toward infrastructure. I repurposed a ThinkPad T430 into a headless Ubuntu Server and used it to learn Linux sysadmin work the way it actually gets done — Docker containers, TLS I had to compile a custom binary to get working, a mesh VPN instead of open ports, and signed commits on every configuration change. Every phase is documented in GitHub with bash setup scripts and config templates, because infrastructure nobody can reproduce isn't really infrastructure.

I'm currently pursuing a B.S. in Cloud and Network Engineering (AWS) at Western Governors University, expected March 2030, while working data center operations at Securitas. I'm open to IT support or cloud roles — remote or hybrid. If you need someone with CRM experience, real Linux practice, and a growing cloud background, let's connect.
